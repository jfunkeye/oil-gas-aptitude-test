# Oil & Gas Aptitude Test

A comprehensive aptitude test practice application for the Nigerian oil and gas industry technical graduate programme.

## 🚀 Features

- **60 Questions** covering 6 sections:
  - Numerical Reasoning
  - Verbal Reasoning
  - Abstract Reasoning
  - Logical Reasoning
  - Engineering Fundamentals
  - Situational Judgement Test (SJT)
- **90-Minute Timer** with auto-submit
- **Multiple Question Sets** - Fresh questions on each retake
- **Detailed Explanations** for every question
- **Results Dashboard** with performance analysis
- **Wrong Answer Review** to learn from mistakes
- **Responsive Design** - Works on all devices
- **Navigation Guard** - Prevents accidental test abandonment

## 🛠️ Tech Stack

- **React 18** - UI Framework
- **Vite** - Build Tool
- **Bootstrap 5** - Styling
- **React-Bootstrap** - Bootstrap Components
- **Bootstrap Icons** - Icon Library
- **Local Storage** - Progress Persistence

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/oil-gas-aptitude-test.git

# Navigate to project directory
cd oil-gas-aptitude-test

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build



📱 Application Screens
Home Screen - Test information and start button

Test Screen - Questions with timer and navigation

Results Screen - Score, performance level, and review

🎯 How It Works
Click "Start Practice Test"

Complete 60 questions within 90 minutes

Submit your answers or let the timer auto-submit

View your results and review wrong answers

Click "Retake Test" for a fresh set of questions

🔄 Question Sets
The application includes 10 unique question sets with 60 questions each:

Set 1 through Set 10

Each retake loads a new set automatically

Progress is saved in your browser's local storage

📊 Performance Levels
Score	Level
80-100%	Excellent - Strong Candidate
67-79%	Good - Likely to Progress
50-66%	Average - Borderline
Below 50%	Needs Improvement
